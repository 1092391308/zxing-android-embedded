Sie können auch stellen Kontaktinformationen in einem QR-Code, und legen Sie sie auf einer Visitenkarte oder Website. Wenn Sie ihn zu scannen, bietet die Ergebnisse Bildschirm eine Auswahl von Aktionen:

Neben URLs und Kontaktdaten können QR-Codes auch enthalten:

  • Kalendereinträge, die Sie in Ihrem Kalender hinzufügen können
  • Telefonnummern, die Sie wählen können
  • SMS-Nummern, die Sie Textnachricht kann
  • E-Mail-Adressen, die Sie per E-Mail
  • Geographische Koordinaten, die Sie in Maps öffnen
  • Plain Text, den Sie lesen können, dann mit einem Freund teilen
